운영 안내

처음 배포할 때 꼭 확인할 세 가지

1. 원본 응답은 공개하지 않기

GitHub Pages에는 기업명, 연락처 같은 원본 응답이 아니라 집계 숫자와 비식별 요약만 노출하세요.

2. 폼 링크부터 먼저 연결하기

사이트 디자인보다도 실제 접수 동선이 동작하는지가 먼저입니다. 모든 버튼을 실제 폼 주소로 교체하세요.

3. 데이터 없을 때 화면도 준비하기

초기 운영 단계에는 숫자가 비어 있는 경우가 많으니, 빈 상태 안내 문구가 보이도록 유지하는 편이 좋습니다.
